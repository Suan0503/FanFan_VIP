# 🚀 快速開始指南

## 立即使用新功能

### 第一步：查看新年選單

在 LINE 群組中輸入：
```
/指令
```
您會看到全新的新年風格管理選單！

### 第二步：設定群組功能

作為主人，您可以控制每個群組的功能：

#### 查看當前功能狀態
```
/功能設定
```

#### 開啟/關閉功能
```
/設定功能 translate     # 翻譯功能
/設定功能 voice         # 語音翻譯
/設定功能 auto_translate # 自動翻譯
/設定功能 statistics    # 統計功能
/設定功能 admin         # 管理功能
```

#### 生成群組 TOKEN
```
/生成token
```

### 第三步：應用場景

#### 免費版（基礎功能）
適合給新用戶試用：
```
# 只保留基本翻譯
/設定功能 voice         # 關閉語音
/設定功能 statistics    # 關閉統計
/設定功能 admin         # 關閉管理
```

#### 標準版（中階功能）
適合一般付費用戶：
```
# 開啟翻譯、語音、自動翻譯
/設定功能 voice         # 開啟語音
/設定功能 statistics    # 關閉統計
/設定功能 admin         # 關閉管理
```

#### 專業版（全功能）
適合高級付費用戶，所有功能都開啟（預設狀態）

| 指令 | 說明 | 誰能用 |

## 💡 實用技巧

### 1. 為不同客戶設定不同方案
```python
# 客戶 A - 免費試用版
/功能設定  # 先查看狀態
/設定功能 voice  # 關閉進階功能
/設定功能 statistics

# 客戶 B - 付費標準版
/設定功能 voice  # 開啟語音功能
/設定功能 statistics  # 保持統計關閉

# 客戶 C - 付費專業版
# 保持所有功能開啟（預設）
```

### 2. 查看並記錄 TOKEN
```
/生成token
```
記下 TOKEN 後可用於：
- API 認證
- 功能驗證
- 客戶管理

### 3. 測試功能限制
讓非主人用戶嘗試使用關閉的功能，會看到友善提示：
```
❌ 本群組未開啟[功能名稱]功能，請聯絡管理員。
```

## 🎯 推薦工作流程

### 新客戶加入
1. 機器人被邀請進群組
2. 輸入 `/功能設定` 查看預設狀態（全開）
3. 根據客戶方案調整功能
4. 輸入 `/生成token` 記錄 TOKEN
5. 告知客戶可用功能

### 客戶升級
1. 客戶要求升級方案
2. 輸入 `/功能設定` 查看當前功能
3. 輸入 `/設定功能 [功能名]` 開啟新功能
4. 通知客戶功能已升級

### 功能維護
1. 定期檢查各群組功能使用狀況
2. 使用 `/統計` 查看使用數據
3. 根據需求調整功能配置

## ⚠️ 重要提醒

1. **TOKEN 保密**: 請勿在公開群組展示 TOKEN
2. **備份資料**: 定期備份 `data.json` 檔案
3. **測試優先**: 在測試群組先測試功能
4. **權限管理**: 只有主人可以修改功能設定

## 🆘 常見問題

### Q: 用戶說無法使用某功能？
A: 使用 `/功能設定` 檢查該群組是否已啟用該功能

### Q: 如何恢復預設狀態？
A: 將所有功能都開啟，或刪除該群組的 feature_switches 設定

### Q: TOKEN 遺失了怎麼辦？
A: 使用 `/功能設定` 查看，或使用 `/生成token` 生成新的

### Q: 可以為多個群組設定相同功能嗎？
A: 可以，每個群組獨立管理，但 TOKEN 都是唯一的
